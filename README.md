## 强度计算规则
- 基础强度：value=basicValue;
- WA增量：value+=WATimes*WAValue;
- AC增量：value+=ACTimes*ACValue;
- 排名增量：value+=(rank-1)*rankValue;
- 强度上限：if (value>limit) v=limit;
